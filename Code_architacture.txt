Ici, nous trouvons l'exemple d'un code Nextflow et d'un container docker

Le process est un docker : 
process reformatFasta {
executor =" local "
container =" evolbioinfo / goalign : v0 .3.2"
input :
file sequences
output :
file "*. phylip "
script :
"""
goalign reformat phylip -i ${ sequences } -o ${ sequences . baseName }. phylip
"""
}
Et là, c'est l'intégration du container dans le worklow codé avec Nextflow
workflow {
fastaFiles = Channel . fromPath ("/ home / user / fasta /*. fasta ")
phylipFiles - reformatFasta ( fastaFiles )
}
